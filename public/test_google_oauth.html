Setup Instructions

1. Configure Google OAuth

  1. Update google-oauth-config.js with your Client ID
  2. Update home.php with your Client ID and Secret
  3. Add your domain to Google Cloud Console authorized origins

2. Run Database Migration

  1. Go to setup_google_oauth.php
  2. Click "Run Migration" to add Google OAuth fields
  3. Verify the migration was successful

3. Test the Integration

  1. Go to home.php
  2. Click "Sign in with Google" or "Sign up with Google"
  3. Complete the OAuth flow
  4. Verify successful login
